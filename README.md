# Twilio Messaging Services Demonstration

Test out Twilio's SMS/MMS, Voice, and WhatsApp APIs with a webapp enabling sending, receiving, and responding with media! :telephone_receiver: :calling:

Using [Bootstrap](https://getbootstrap.com/), [Font Awesome 5](https://github.com/FortAwesome/Font-Awesome) for visuals, with webapp running in [Flask](https://github.com/pallets/flask).
